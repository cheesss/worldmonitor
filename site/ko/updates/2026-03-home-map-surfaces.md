# 2026-03: 앱 수준 지구본과 2D 지도를 메인 페이지로 이동

## 무엇이 바뀌었나

- 홈의 경량 지도 위젯을 앱 수준 3D 지구본으로 교체했습니다.
- 데스크톱 스타일의 2D 운영 지도를 홈에 직접 추가했습니다.
- `showcase/globe` 전용 경로는 선택형 집중 뷰로 유지했습니다.
- 기능 인덱스도 홈을 첫 번째 시각 진입점으로 다시 정리했습니다.

## 왜 중요한가

방문자가 별도 쇼케이스 경로로 이동하지 않아도 메인 페이지에서 바로 실제 제품에 가까운 시각 표면을 확인할 수 있습니다.

## 사용자 영향

- 홈에서 즉시 더 풍부한 3D 지구본을 회전하며 볼 수 있습니다.
- hotspot, 관계선, 경로, 밀도 레이어가 있는 2D 지도도 바로 눌러볼 수 있습니다.
- 문서 홈이 정적 랜딩 페이지보다 제품 표면에 더 가까운 역할을 하게 됐습니다.
